# CCNA Guide

An educational web application designed to teach networking fundamentals and prepare students for the CCNA 200-301 certification.

## Features

- **Beginner-to-Pro Path:** Structured modules starting from basic computer components.
- **Accurate Content:** All data is derived from official Cisco CCNA source materials.
- **Interactive Quizzes:** Knowledge checks after each module with real-world exam questions.
- **Modern UI:** Clean, responsive design inspired by Cisco's aesthetic.

## Tech Stack

- **React (TypeScript)**
- **Vite**
- **Vanilla CSS**
- **React Router**

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Deployment

To deploy to GitHub Pages:

1.  Ensure you have a GitHub repository named `ccna-guide`.
2.  Run:
    ```bash
    npm run deploy
    ```
